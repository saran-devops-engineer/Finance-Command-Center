# Finance Command Center

A mobile-first, local-first financial decision support PWA for individuals in India.

The product is not an expense tracker. It is designed to answer three questions quickly:

- Am I okay?
- What needs my attention?
- What should I do next?

## V1 Direction

- Next.js App Router with strict TypeScript.
- Tailwind CSS and shadcn-style reusable components.
- Local-first architecture with IndexedDB as the source of truth.
- Rule-based financial health and recommendations.
- No authentication, backend, cloud storage, or AI APIs in V1.
- PWA-ready metadata and installable app shell.

## Current Focus

The first milestone validates the core product flow with demo data:

- Home dashboard.
- Premium loan experience.
- Money, Insights, and Profile navigation.
- Financial snapshot and recommendation engines separated from UI.
