# Roadmap

## V1 Goal

Validate the product vision, core navigation, local-first privacy model, and decision-support flow.

V1 should not try to become a complete personal finance suite.

## V1 Milestones

### Foundation

- Next.js 15 app.
- TypeScript.
- Tailwind CSS.
- Mobile shell.
- PWA metadata.
- GitHub Actions CI.

### Local-First Data

- IndexedDB persistence.
- Repository boundary.
- Guided onboarding.
- Local profile.
- Local money breakdown.
- Local loans.
- Local payment history.

### Core Decision Flow

- Home financial status.
- Available money.
- Today's best decision.
- Rule-based recommendations.
- Upcoming dues.
- Active portfolio summary.

### Loans

- Add loan.
- Loan overview cards.
- Loan detail.
- Prepayment simulation.
- Log EMI/prepayment/part-payment.
- Payment history.

### Money

- Cash-flow summary.
- Edit income and commitments.

## V1.1

- Encrypted `.fcc` export.
- Encrypted `.fcc` restore.
- Backup and restore UX.
- Edit/delete loans.
- Better onboarding validation.
- Recent activity on Home.
- PWA offline service worker.

## V2

- Apple Login.
- Google Login.
- Email Login.
- Cloud backup.
- Multi-device sync.
- User-controlled restore.

## V3

- AI financial advisor.
- Push notifications.
- Family accounts.
- Investment tracking.
- Advanced analytics.
- Scenario planning.

## Product Principle

Advance the roadmap only when the feature helps the user make a better financial decision.
