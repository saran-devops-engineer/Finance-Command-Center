# Loans Prompt

Use this prompt when improving Loans.

## Screen Question

Which loan needs attention?

## Goals

- Help users understand debt burden.
- Highlight highest-interest loans.
- Show upcoming EMI risk.
- Make prepayment decisions easier.
- Keep loan overview premium and card-based.

## Rules

- Do not use tables for loan overview.
- Preserve premium card feel.
- Keep calculations in services.
- Label projections as estimates.
- Prefer clear decisions over raw analytics.

## Good Improvements

- Edit/delete loan.
- Pin loan.
- Better loan priority logic.
- Payment history detail.
- Better prepayment simulation.
- Friendly date formatting.

## Avoid

- Overloading cards with every field.
- Unexplained financial advice.
- Bank-grade claims without verified formulas.
