# Home Prompt

Use this prompt when improving the Home Screen.

## Screen Question

Am I financially okay today?

## Goals

- Show financial status clearly.
- Explain available money after mandatory commitments.
- Surface today's best decision.
- Show urgent dues.
- Show at most three active portfolio loans.
- Always include View All Loans.

## Rules

- Do not redesign the visual language.
- Preserve whitespace.
- Keep Apple-inspired aesthetics.
- Use contextual messages, not generic dashboard labels.
- Keep business logic in helpers or services when it grows.
- Do not show every loan on Home.

## Good Improvements

- Better decision copy.
- Better due-date prioritization.
- Recent Activity from payment history.
- Better empty states.
- Friendly date formatting.

## Avoid

- Tables.
- Dense charts.
- Showing every financial object.
- Calling rule-based recommendations real AI.
