# Purpose

This document defines every business concept used by the Home Loan Engine.

These definitions are the single source of truth.

No implementation may redefine these concepts.

---

# Philosophy

A Home Loan is not just money borrowed.

It is a long-term financial contract that changes over time.

The Home Loan Engine models this contract mathematically.

Every calculation, recommendation and simulation must respect these business definitions.

---

# Entity

## Home Loan

Definition

A secured loan provided for purchasing, constructing or renovating residential property.

The borrower repays the loan through scheduled installments over an agreed tenure.

# Principal

Definition

The original amount borrowed.

Principal never changes.

Outstanding Principal changes.

---

# Outstanding Principal

Definition

Remaining unpaid principal.

Every payment reduces this value.

Interest never increases Outstanding Principal unless specifically supported by lender rules.

---

# EMI

Definition

Equated Monthly Installment.

Every EMI consists of

Interest Component

+

Principal Component

The ratio changes every month.

---

# Interest Rate

Definition

Annual borrowing rate.

Expressed as percentage.

Future versions support

Floating

Fixed

Hybrid

Step-up

Step-down

---

# Remaining Tenure

Definition

Number of installments remaining.

Reduce Tenure simulations modify this value.

Reduce EMI simulations do not.

---

# Payment

A payment represents money applied toward the Home Loan.

Types

Regular EMI

Part Prepayment

Foreclosure

Penalty

Adjustment

Refund

---

# Part Prepayment

Definition

Additional payment made toward principal before loan closure.

Purpose

Reduce

Interest

Tenure

or

EMI

depending on simulation.

---

# Foreclosure

Definition

Complete repayment of the outstanding principal.

Loan Status becomes Closed.

No future EMI exists.

---

# Amortization Schedule

Definition

The complete month-by-month repayment schedule.

Every month records

Opening Balance

Interest

Principal

Closing Balance

EMI

This schedule is the authoritative source for all future calculations.

Shortcut formulas must never replace the amortization schedule for simulation results.

---

# Loan Progress

Definition

Percentage of principal repaid.

Not

Percentage of EMIs paid.

Users care about debt reduction,

not payment count.

---

# Loan Status

Supported States

Draft

Active

Paused

Foreclosed

---

# Loan Health

Loan Health is an assessment generated by the Recommendation Engine.

It is not stored.

Inputs may include

Interest Burden

Outstanding Balance

Payment Behaviour

Cash Flow

Emergency Buffer

Upcoming Dues

---

# Simulation

Simulation creates a temporary copy of the Home Loan.

No simulation modifies actual loan data.

Every simulation compares

Current Loan

↓

Scenario

↓

Projected Loan

↓

Difference

↓

Recommendation

---

# Recommendation

Recommendations never belong to the Home Loan itself.

Recommendations belong to the Recommendation Engine.

The Home Loan only provides inputs.

---

# Final Rule

Every future Home Loan feature must use these definitions.

If a new concept is introduced,

it must first be added to this Domain Model before implementation.
